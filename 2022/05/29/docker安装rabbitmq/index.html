Docker安装Rabbitmq

拉取镜像

docker pull rabbitmq:3.8.8 后面不选择版本默认是最新版

启动镜像

开放端口

开放服务器5672、15672两个端口

安装web管理页面

此时可以使用服务器ip:15672 进行访问，默认用户名密码都为guest

安装延时插件

  1. 下载插件：下载地址

  2. 将插件传输到linux系统中，我是使用xftp上传到 /opt目录下

  3. 安装插件

  4. 进入管理页面，看到type有x-delayed-message则成功安装延时插件
