docker安装redis

首先在宿主机创建一个redis目录，然后将默认的redis.conf文件复制到该目录下。如果没有，可以使用下面提供的配置。然后修改redis.conf目录下的配置，主要修改：bind，注释或者改为允许连接的ip地址，一般设置为0.0.0.0；requirepass 设置密码；·daemonize修改为no，不是后台启动，否则与docker run -d 冲突。

然后使用下面的命令启动redis

redis-server /etc/redis/redis.conf表示读取redis.conf文件启动redis，由于容器内的redis.conf文件和宿主机的关联起来，因此读取的是宿主机的配置文件。
