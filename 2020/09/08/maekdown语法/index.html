markdown语法

发表于 2020-09-08 分类于 markdown

一、标题

在想要设置为标题的文字前面加#来表示

一个#是一级标题，两个#是二级标题，以此类推。支持六级标题

并且在#后面加上空格之后在添加文字

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

效果如下

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

二、字体

  • 加粗

    要加粗的文字左右分别使用两个*号包起来

  • 斜体

    使用一个*包起来

  • 斜体加粗

    使用三个*号包起来

  • 删除线

    使用两个~~号包起来

**加粗**
*斜体*
***斜体加粗***
~~删除线~~

效果如下

加粗
斜体
斜体加粗
删除线

三、引用

在引用的文字前面加>即可，也可以嵌套使用，如加两个>,三个>,n个>

>引用
>>引用2
>>>引用3

效果如下

引用

引用2

引用3

四、分割线

三个或三个以上的+ - *都可以

+++
*****
----

五、图片

![图片alt](图片地址 "图片title")
图片alt就是显示在图片下面的文字，相当于对图片内容的解释
图片title是图片的标题，当鼠标移到图片上时显示的内容，title可加可不加

六、超链接

[超链接名](超链接地址 "超链接title")
title 可加可不加
[百度] (http://www.baidu.com)

效果如下：

百度

注意：markdown本身不支持超链接在新页面中打开，如果想在新页面中打开的话可以用html中的a标签来代替

<a href="超链接地址" target="_blank">超链接名</a>

效果如下

百度 (但是这个好像也打开不了)
